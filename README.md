# WAWP Forecast Engine

Python pipeline for WAWP aviation forecast guidance, model verification, quantile mapping, and dashboard publication.

The current TITAN overhaul adds Open-Meteo ingestion, AWOS hourly and 1-minute foundations, multi-parameter quantile mapping, diurnal climatology, CAPE-aware TSRA detection, and ICAO TAF safety fixes.

See [docs/README.md](docs/README.md) for architecture notes.
